\title{A Minimal Conformance Article for the WasmTeX Engine}

\author{Ada Testwright}
\address{WasmTeX Conformance Corpus}

\begin{abstract}
This original document verifies that WasmTeX compiles an article written for the
elsarticle class when that class is supplied to the compiler as an input file
rather than shipped in a bundle. It exercises the elsarticle front matter,
sectioning, and display mathematics.
\end{abstract}

\begin{keyword}
WebAssembly \sep TeX \sep conformance \sep typesetting
\end{keyword}

\end{frontmatter}

\section{Introduction}
The WasmTeX engine loads the elsarticle class from the job's file set. The
class requests packages that live in the on-demand academic tier, which the
engine mounts automatically before the first pass, so the supplied class
typesets without that tier having to be preloaded for it.

\section{A Displayed Equation}
A displayed identity provides a rendering check:
\begin{equation}
\sum_{n=1}^{\infty} \frac{1}{n^{2}} = \frac{\pi^{2}}{6}.
\end{equation}

\section{Conclusion}
A supplied elsarticle class typesets end to end under WasmTeX, demonstrating the
ability to compile top-journal templates without bundling their classes.
