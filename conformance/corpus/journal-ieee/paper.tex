\begin{abstract}
This original document verifies that WasmTeX compiles a paper written for the
IEEEtran class when that class is supplied to the compiler as an input file
rather than shipped in a bundle. It exercises the abstract, keyword block,
sectioning, and inline and display mathematics.
\end{abstract}

\begin{IEEEkeywords}
WebAssembly, TeX, conformance, typesetting
\end{IEEEkeywords}

\section{Introduction}
The WasmTeX engine loads the IEEEtran class from the job's file set. IEEEtran
typesets in Times and this paper loads the \texttt{cite} package, both of which
live in the on-demand academic tier; the engine mounts that tier automatically
before the first pass, so the supplied class typesets without any tier having to
be preloaded for it.

\section{A Displayed Equation}
The Gaussian integral provides a compact rendering check:
\begin{equation}
\int_{-\infty}^{\infty} e^{-x^{2}}\, dx = \sqrt{\pi}.
\end{equation}
An inline fragment, $E = mc^{2}$, confirms math-mode kerning as well.

\section{Conclusion}
A supplied IEEEtran class typesets end to end under WasmTeX~\cite{wasmtex},
demonstrating the ability to compile top-journal templates without bundling
their classes.
